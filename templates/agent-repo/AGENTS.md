# Agent Repository — Codex Instructions

This repository contains one immutable experiment for Fieldwork's "30 Agents in 30 Days" project.

Before implementation, read:

- `brief.md`
- `README.md`
- `evaluation.md`
- `field-notes.md`
- `promotion-review.md`

Also follow the programme guidance in the central `30-agents` repository:

- `docs/project-charter.md`
- `docs/engineering-principles.md`
- `docs/evaluation-framework.md`
- `docs/content-strategy.md`

## Rules

- Build one narrow agent or agentic workflow.
- Test one hypothesis.
- Do not expand scope after implementation starts.
- Preserve the brief once coding starts.
- Evaluate before polish or publication.
- Capture evidence for the verdict.
- Do not publish sensitive data, credentials, customer information or internal operational details.
- Do not promote reusable work without completing `promotion-review.md`.
- Leave the experiment understandable to another engineer.
